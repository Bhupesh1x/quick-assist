# Quick Assist: A B2B AI SaaS Support Platform

## Description
The **B2B AI SaaS Support Platform** is a full-stack, AI-powered customer support solution built from scratch using **Next.js 15**, **React 19**, **Convex**, **Turborepo**, and **Vapi**.

This platform enables businesses to deploy a real-time AI support agent that can answer customer questions, reference a company-specific knowledge base, and intelligently escalate conversations to human operators when needed. With workspace management, authentication, billing, and an embeddable widget, this project provides everything required to launch a modern B2B AI customer support SaaS.

Businesses can:

- Train the AI using documentation, uploaded files, embeddings, and RAG.
- Provide support via **real-time chat** and **voice**.
- Automatically resolve issues or escalate to a human operator.
- Use the **Integrations** section to generate a simple script tag that can be added to any websiteâ€”instantly activating the support widget for end users.
- Manage users, teams, roles, and billing with **Clerk**.
